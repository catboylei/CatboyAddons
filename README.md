## What does CatboyAddons do ?

CatboyAddons adds meowing back to skyblock, and other qol features mainly to replace neu for dungeon players

- Global woof toggle

## Installation Guide

1. **Install [ChatTriggers](https://chattriggers.com/) for Minecraft 1.8.9**
2. **Download the latest `CatboyAddons.zip`**  
   [Releases Page](https://github.com/catboylei/CatboyAddons/releases)
3. **Extract the `.zip` file** into your `modules` folder:  
   `/.minecraft/config/ChatTriggers/modules/` or use /ct files in game
5. Launch Minecraft and run `/ct reload` in chat.
6. Use `/cba` to open the CatboyAddons menu.

---

## **What to do if it breaks ?**  

1. **Make sure you have the latest ChatTriggers version installed**
2. **Make sure you have the dependencies installed**
   `/ct import amaterasu`
   `/ct import pogdata`
3. **Run** `/ct reload`
4. Message @catboylei on discord with relevant info
