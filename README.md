# CatboyAddons

**The [ChatTriggers](https://chattriggers.com/) module for Hypixel SkyBlock Meowers**  

## Download

[Visit Releases Page](https://github.com/catboylei/CatboyAddons/releases)

---

## Features

### General

- Global woof toggle
- Global mod toggle

### Meowing

- Configurable auto meow
- Meow sounds
- Meow speak
- Configurable meow filter
- Configurable random meows
- cat inventory

### Inventory

- Slot binding
- Slot locking
- Wardrobe keybinds
- Inventory Buttons

### Notifiers

- Blood rush titles
- Boss titles

## Installation Guide

1. **Install [ChatTriggers](https://chattriggers.com/) for Minecraft 1.8.9**
2. **Download the latest `CatboyAddons.zip`**  
   [Releases Page](https://github.com/catboylei/CatboyAddons/releases)
3. **Extract the `.zip` file** into your `modules` folder:  
   `/.minecraft/config/ChatTriggers/modules/` or use /ct files in game
5. Launch Minecraft and run `/ct reload` in chat.
6. Use `/cba` to open the CatboyAddons menu.

---

## **What to do if it breaks ?**  

1. **Make sure you are using the latest ChatTriggers version** [Chattriggers Releases](https://github.com/ChatTriggers/ChatTriggers)
2. **Make sure the dependencies are installed**  
   `/ct import amaterasu`  
   `/ct import pogdata`  
   `/ct import bloomcore`
4. **Run** `/ct reload`
5. if it still does not work after doing the above, message @catboylei on discord with relevant info
